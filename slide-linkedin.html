MyBank — Demo Banking App

Lightweight Node/Express backend + static frontend — ready for demos and API testing
Live demo
https://mybank-8s6n.onrender.com

What it shows

Authentication (JWT), CRUD for transactions, server logging and responsive UI (desktop + mobile).

Key points

Ready to test using Postman / curl / PowerShell — server persists data to data.json.

How to demo

Open the live URL → register/login → add a transaction → show table update and server logs (no install required).
